# Taskera – Full Stack Task Management Application

This project is a Trello-like task management application developed to strengthen our CVs and gain hands-on experience in full-stack team collaboration.

The project is structured as a monorepo and contains both the frontend and backend applications in separate folders within the same repository.

## 🚀 Tech Stack

### Frontend

- Next.js
- TypeScript
- Tailwind CSS
- REST API Integration

### Backend

- ASP.NET Core Web API
- Entity Framework Core
- JWT Authentication
- PostgreSQL / SQL Server

## 🎯 Project Goal

The goal of this project is to build a Kanban-style task management system where users can:

- Register and log in
- Create and manage boards
- Create lists inside boards
- Create and manage cards inside lists
- Move cards using drag & drop
- Access data based on user authorization

## ⚙️ Development Setup

### 📌 Run Frontend

- cd frontend
- npm install
- npm run dev

### 📌 Run Backend

- cd backend
- dotnet run
